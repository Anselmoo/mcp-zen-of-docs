# Prompt Templates

This directory contains **`.prompt.md`** files — reusable prompt
templates for GitHub Copilot Chat.

## What Are Prompt Files?

Each `.prompt.md` file defines a reusable prompt that appears in
the Copilot Chat slash-command menu. Prompts support three modes:

| Mode    | Behaviour                                          |
| ------- | -------------------------------------------------- |
| `ask`   | Read-only — answers questions without editing files |
| `edit`  | Applies inline edits to the current file            |
| `agent` | Full agentic mode with tool access                  |

## Required Frontmatter

```yaml
---
agent: 'agent'            # ask | edit | agent
tools: ['read', 'agent', 'edit', 'search', 'web', 'zen-of-docs/*', 'context7/*', 'serena/*', 'todo']
description: 'Short description shown in command palette'
---
```

## Conventions

- Keep prompt files deterministic and reviewable.
- Prefer additive prompt evolution over destructive rewrites.
- Use descriptive filenames: `init-docs.prompt.md`, `review-nav.prompt.md`.
